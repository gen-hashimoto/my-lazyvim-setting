## my setting on Ubuntu

- for Python formatting

```bash
sudo apt install python3.12-venv
```

- for Mason

```bash
sudo apt install nodejs npm
```

- to paste text copied from Neovim

```bash
sudo apt install xsel
or:
sudo apt install xclip

```
